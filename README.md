# 📚 AI Study Planner

## 🚀 Overview
This is a simple AI-based Study Planner and Productivity Tracker built using Python.

## ✨ Features
- Generate study timetable
- Track daily study hours
- Visualize data using graphs
- Basic AI-based suggestions

## 🛠 Tech Stack
- Python
- Streamlit
- Pandas
- Matplotlib

## ▶️ How to Run
1. Install dependencies:
   pip install streamlit pandas matplotlib

2. Run the app:
   streamlit run app.py
